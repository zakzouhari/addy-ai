Welcome to Clearpath Processor
A Clearpath Automation, LLC product

Here's everything you can do with Clearpath Processor:

Import MISMO XML
Drop a MISMO 3.x file into Loan Pipeline → automatically creates a loan with borrower data pre-filled and a standard missing-items checklist.
AI Loan Analysis
Open any loan → click AI Analyze to get a file completeness score, prioritized missing items list, and red-flag summary — powered by Claude AI.
Multi-Channel Outreach
Use Send Reminder to reach borrowers via Email, SMS (Twilio), or AI Voice Call (ElevenLabs) — all from one click.
Submit to Underwriter
Generate an AI cover letter + submit the full package to Simple Nexus, directly to the underwriter, or download a bundle manifest.
Quick start: Click Create Sample Loan on the Dashboard to see a fully-populated demo file — then explore AI Analyze, Send Reminder, and Submit to UW.
